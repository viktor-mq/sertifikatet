📂 Import/Export Spørsmål

📥 Eksporter Spørsmål

Last ned alle spørsmål som CSV eller JSON for backup eller redigering.

📤 Importer Spørsmål

Last opp en CSV- eller JSON-fil med spørsmål. Filen må ha samme format som eksport-filen.

📁 Aksepterte formater: CSV (.csv) og JSON (.json)
💡 I Norge brukes vanligvis semikolon (;) som skilletegn

📄 Filformat:

CSV-kolonner: id, question, option_a, option_b, option_c, option_d, correct_option, category, subcategory, difficulty_level, explanation, image_filename

🇳🇴 Norsk format: id;question;option_a;option_b;...

🌍 Internasjonalt: id,question,option_a,option_b,...

Tips: Last ned en eksport-fil først for å se det korrekte formatet.

JSON-format: En liste med spørsmål-objekter med samme felt som CSV

Eksempel: [{"id": 1, "question": "...", "option_a": "...", ...}]

Tips: Eksporter som JSON først for å se det korrekte formatet.
